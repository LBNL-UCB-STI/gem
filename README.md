## Typical git workflow

Pull the latest:

```bash
git pull
```

Work on your code.

Stage your changes:

```bash
git add file1.R
git add file2.R
git add manyFiles*
```

Commit the staged files:

```bash
git commit -m "one sentence description of what changed"
```

Push all commits back to github:

```bash
git push
```
